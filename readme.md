# Hate Speech Detection Chatbot

A Flask-based chatbot that detects hate speech using a fine-tuned DistilBERT model.

## Features

- Multi-label hate speech detection
- Hate category identification
- Severity classification
- Violation tracking
- Temporary user blocking
- Messenger-style interface (in progress)

## Tech Stack

- Python
- Flask
- PyTorch
- Transformers
- SQLite
- HTML/CSS/JavaScript
